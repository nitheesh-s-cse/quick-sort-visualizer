Graph Tree Explanation

This visualization represents recursion as a 3D graph tree.

  • Each recursive call is shown as a node.
  • Each partition creates left and right recursive branches.
  • Left subtree = elements < pivot
  • Right subtree = elements ≥ pivot

Click a node to focus the camera on that subtree.

Pseudocode

function quickSort(arr, low, high):
  if low < high:
    pivotIndex = randomPartition(arr, low, high)
    quickSort(arr, low, pivotIndex - 1)
    quickSort(arr, pivotIndex + 1, high)

function randomPartition(arr, low, high):
  randomIndex = random(low, high)
  swap(arr[randomIndex], arr[high])
  return partition(arr, low, high)

function partition(arr, low, high):
  pivot = arr[high]
  i = low - 1
  for j = low to high-1:
    if arr[j] < pivot:
      i++
      swap(arr[i], arr[j])
  swap(arr[i+1], arr[high])
  return i + 1
